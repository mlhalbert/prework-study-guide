Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes are always defined in the beginning element.
  • The base element defines the base URL for a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The three ways to style a webpage using CSS are inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Declarations contain the CSS property that is to be applied and the value of the property.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds all modifications in the current working branch to the staging area.
  • git commit -m "commit-details": creates a commit that captures a snapshot of the currently staged changes.
